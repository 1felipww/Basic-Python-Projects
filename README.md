# Basic Python Projects 🐍

This repository contains a collection of my four fundamental Python projects. These scripts are perfect for beginners and demonstrate the core concepts of programming, such as control flow, user input, loops, and the use of built-in libraries.

## 💻 Projects Included

### 1. Simple Calculator (`calculator.py`)
A command-line application that performs basic arithmetic operations.
* **Features:** Allows the user to choose an operation (addition, subtraction, multiplication, or division) and input numbers to get an instant result.

### 2. Dice Simulator (`dice_simulation.py`)
A fun little script that mimics the rolling of a die.
* **Libraries used:** `random` (built-in)
* **Features:** Randomly generates a number between 1 and 6, simulating a real dice roll. Great for learning how random number generation works in Python.

### 3. Number Guessing Game (`number_guessing.py`)
An interactive command-line game where the computer hides a number, and the user has to guess it.
* **Libraries used:** `random` (built-in)
* **Features:** Provides hints (like "too high" or "too low") based on the user's input until the correct number is guessed. 

### 4. Quote Generator (`quote_generator.py`)
A simple script that outputs a random inspiring or interesting quote every time it is run.
* **Features:** Selects and prints a random text string from a pre-defined collection.

---

## ⚙️ Prerequisites & Setup

All of these projects use pure Python and rely solely on standard built-in libraries. **No external installations or modules are required.**

# Projetos Básicos em Python 🐍

Este repositório contém uma coleção dos meus quatro projetos fundamentais em Python. Estes scripts são perfeitos pra iniciantes e mostram os conceitos centrais da programação, como fluxo de controle, entrada do usuário, loops e o uso de bibliotecas nativas.

## 💻 Projetos Inclusos

### 1. Calculadora Simples (`calculator.py`)
Um aplicativo de linha de comando que realiza operações aritméticas básicas.
* **Funcionalidades:** Permite ao usuário escolher uma operação (adição, subtração, multiplicação ou divisão) e inserir números para obter um resultado instantâneo.

### 2. Simulador de Dados (`dice_simulation.py`)
Um script divertido que imita o lançamento de um dado.
* **Bibliotecas usadas:** `random` (nativa)
* **Funcionalidades:** Gera aleatoriamente um número entre 1 e 6, simulando o lançamento de um dado real. Excelente para aprender como funciona a geração de números aleatórios em Python.

### 3. Jogo de Adivinhação de Números (`number_guessing.py`)
Um jogo interativo de linha de comando onde o computador esconde um número e o usuário precisa adivinhá-lo.
* **Bibliotecas usadas:** `random` (nativa)
* **Funcionalidades:** Fornece dicas (como "muito alto" ou "muito baixo") com base no palpite do usuário até que o número correto seja adivinhado. 

### 4. Gerador de Citações (`quote_generator.py`)
Um script simples que exibe uma citação inspiradora ou interessante aleatória toda vez que é executado.
* **Funcionalidades:** Seleciona e imprime uma string de texto aleatória a partir de uma coleção predefinida.

---

## ⚙️ Pré-requisitos e Instalação

Todos esses projetos usam Python puro e dependem exclusivamente de bibliotecas nativas padrão. **Nenhuma instalação ou módulo externo é necessário.**
